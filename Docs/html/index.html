Color Golems

Color Golems is a simple Genetic Algorithm approach to generate multiple colored mobs.

  • Developed by Renato Avellar Nobre - 984405
  • Class Project: Artificial Inteligence for Video Games 2021-2022
  • Università degli Studi di Milano

Motivation

This project presents Color Golems a simple version of the GOLEM (GOLEM – Generator Of Life Embedded into MMOs) - an algorithm aimed at increasing variety and unpredictability among mobs in MMOs, by means of an approach rooted into Genetic Algoritms (GAs).

Color Golems aim to create a wide variety of colorfull creatures using Genetic Algorithms and the DeltaE2000 for comparing color similarities.

Features

  • Procedural Generated Terrain
  • Simple Player for Game View
  • Genetic Algorithm for Population Diversity
  • Diverse Golems
  • Simple UI for Population Control
  • DeltaE2000 Fitness
  • Doxygem Documentation

Play Instructions

  • WASD for Movement
  • Mouse Movement to Look
  • Space to Jump
  • M to Toggle Genetic Algorithm UI
